# 311ComputerArchitecture

This repository currently contains course-related and research-oriented materials.

## Included Research Workspace

One of the main research folders in this repository is:

- `Desktop/adaptive_time_quantum_aging_papers/`

This workspace focuses on a scheduler research idea at the intersection of:

- adaptive time quantum selection
- hardware aging and reliability-aware control
- AI-assisted scheduling decisions

## Main Research Question

Can a scheduler select `time quantum` values not only from queue-state information, but also from
thermal and aging-related hardware feedback, in order to achieve:

- the same performance with less aging, or
- better performance under the same aging budget?
